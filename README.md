## 🔍 Troubleshooting Issues

Most problems result from incorrect settings or network blocks. Try these steps if the app fails to start:

*   Check your internet connection first.
*   Verify your API key has enough credit.
*   Ensure your firewall allows Claudex to access the internet.
*   Close other programs that might compete for memory if the app feels slow.

## 🛡️ Privacy and Data

Claudex prioritizes your security. The application keeps your API keys on your local machine. It does not send your personal data to unknown servers. Only the necessary requests reach the model providers. You retain full control over the information you share.
